# Insurance UI Review

AGENTS.md already covers global frontend and domain rules. Use this concise review checklist.

## Check

- Mobile-first layout, spacing, wrapping, sticky header behavior, and CTA visibility.
- CTA clarity and whether the next step is honest and specific.
- Loading, error, success, empty, and partial-success states.
- Labels, landmarks, visible focus, keyboard operation, and accessible link/button text.
- Trust content: product scope, eligibility, documents, territory, terms, restrictions, support contacts, and privacy expectations.
- Tone: no fake urgency, aggressive sales language, vague promises, or unsupported claims.
- Visual consistency with the existing restrained Green Card Agency CSS system.

## Output

Give concrete component/page-level findings or fixes, not generic UI advice. Report verification performed and any claim/accessibility risks.
