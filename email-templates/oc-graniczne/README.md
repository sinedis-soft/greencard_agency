# OC Graniczne email templates

HTML templates for Bitrix24 automation emails. The files keep Bitrix placeholders such as `{=Document:CONTACT.NAME}` unchanged so they can be copied into CRM email actions.

Templates:

- `01-application-received.html` — application accepted for processing.
- `02-sent-to-insurer.html` — data sent to insurer for acceptance.
- `03-approved-payment.html` — application approved and payment instructions.
- `04-policy-issued.html` — policy issued and attached.
- `05-insurance-agent-info.html` — insurance contract and agent disclosure.
